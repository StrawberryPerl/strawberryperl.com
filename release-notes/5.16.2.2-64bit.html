Strawberry Perl (5.16.2.2-64bit) Release Notes

Released: Feb 24 2013

Check out what is new, what known issues there are, and frequently asked questions about this version of Strawberry Perl. As always, you're encouraged to tell us what you think.

What's new in this Strawberry Perl release: Collapse

  • Mostly bugfix release
  • Fixes for RT#83368, RT#82837, RT#82291
  • Libraries: updates - gmp-5.1.1, libssh2-1.4.3, libxslt-1.1.28 (IMPORTANT!), openssl-1.0.1e
  • Modules: Any::Moose replaced with Moo
  • Startmenu: link to http://win32.perl.org/ replaced with http://metacpan.org

Known issues: Collapse

  • The 64-bit version does not contain all of the modules that the 32-bit version does - Math::Pari (#52146) + all dependants (e.g. Module::Signature).
