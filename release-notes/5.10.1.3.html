Strawberry Perl Release Notes

Version 5.10.1.3, released 29 Jul 2010

Check out what is new, what known issues there are, and frequently asked questions about this version of Strawberry Perl. As always, you're encouraged to tell us what you think.

What's new in Strawberry Perl 5.10.1.3: Collapse

A more polished installer, including link to README file on start menu and in installer, and a start menu that is split into categories.

Added Alien::Tidyp

and Net::SSH2 to Strawberry.

CPANPLUS now uses a directory within the Strawberry Perl installation, instead of within the root directory.

The installer now specifically mentions that Strawberry Perl 5.10.x is hard-coded, and specifies where to go to get a non-hardcoded (5.12.x) version.
