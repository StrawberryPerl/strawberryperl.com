Strawberry Perl (5.14.3.1-32bit) Release Notes

Released: Oct 18 2012

Check out what is new, what known issues there are, and frequently asked questions about this version of Strawberry Perl. As always, you're encouraged to tell us what you think.

What's new in this Strawberry Perl release: Collapse

  • The new perl core 5.14.3
  • Switch to gcc-4.6.3, credits go to Mark Dootson's http://perlmingw.sf.net/
  • Updated all external libraries (openssl, graphics etc.)
  • Added gfortran + pthreads library (PDL)
  • Added new modules: App::cpanminus, Moose + many others (the same set as in 5.16 series)

Known issues: Collapse

  • Do not intall the MSI over the 5.14.2.1 installation as automatic uninstallation does not work correctly. Uninstall the old version manually + remove all residuals (e.g. delete c:\strawberry directory) before installing this version.
