Strawberry Perl Release Notes

Version 5.14.2.1-32bit, released 22 Nov 2011

Check out what is new, what known issues there are, and frequently asked questions about this version of Strawberry Perl. As always, you're encouraged to tell us what you think.

What's new in Strawberry Perl 5.14.2.1-32bit: Collapse

Currently the latest perl-5.14.2

Upgraded gcc toolchain + external libraries + perl modules installed on top of perl core.

Cosmetic changes concerning Start Menu + README.txt

Improved updating of PATH environment variable during MSI installation.

Huge improvements and fixes in 32bit portable edition.


Known issues: Collapse

  • MSI installer: Update of PATH environment variable was improved however it still might happen that after installing MSI you will need to logout/login.
  • Under some (quit rare) circumstances the gcc-4.4.7 may generate 32bit code that causes perl.exe to crash.
